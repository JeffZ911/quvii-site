FOLDER CONTENTS
  00_Masters/        Core wordmark + Q symbol, transparent PNG (red / white / black)
                     + red-on-white JPG master.
  01_Square_1to1/    512 & 1024 px — app icon tile, circle badge, vertical lockup.
  02_Header_4to1/    2000×500 px — site header / signboard (on white, ink, red, left-aligned).
  03_Vertical_2to3/  1200×1800 px — hangtag / vertical poster (on white, red, ink).
  04_Landscape_3to2/ 1800×1200 px — ad / detail-page logo plate.
  05_Mini/           32 & 80 px favicon tiles + 80 px Q watermark (red / white).

COLORWAYS
  Original (red) and Reversed (white) are provided throughout; a one-color
  black master is in 00_Masters for single-color print.

USAGE
  • Keep clear space equal to the wordmark height on all sides.
  • Minimum wordmark width 200 px / 20 mm; below that use the Q symbol.
  • Red is an accent, not a flood — aim for ~10% red, 90% neutral.
  • Never stretch, recolor, rotate, or add effects to the logo.

NOTE
  These are production raster (PNG/JPG) exports. Editable vector masters
  (AI / EPS / SVG) should be produced from the source artwork for print,
  signage and embroidery. The full guideline document is in the deck.
